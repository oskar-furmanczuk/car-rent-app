1) Launching

In order to launch the application you will need:
- Java 11 installed and configured 
- configured PostgreSQL database (according to the information in the "application.properties" file)

To run the application open the command-line and in the catalog with the pom.xml file call
java -jar .targetcar-rent-app-0.0.1-SNAPSHOT.jar


2) Description of the application

The application allows to manage a car rental company. Through the "Customers List" tab we can edit, add and delete customers.
The car rental functionality is also provided via this tab. A maximum of one car is allowed per customer.
The tab "Cars List" enables you to add, edit and delete cars.
